# Kushy Chatbot

The (world's first?) cannabis chatbot. Uses the Kushy API to serve cannabis data (strains, products, shops, etc).

## Development

### Start development

`npm start`

### Build production files

`npm run build`

### Etc

See create-react-app-README for more details.
